# 4-D 레퍼런스 매트릭스

레퍼런스 수집은 단일 차원 X. 4가지 *목적*별로 분리해서 채운다. 각 슬롯은 **다른 출력**을 만든다.

| 슬롯 | 무엇을 찾나 | 산출물 |
|---|---|---|
| **타겟 visual diet** | 타겟이 평소 *일상에서 노출되는* 사이트/앱 | 시각 톤·색감·typography 친숙함 |
| **같은 도메인 강자** | 직접 경쟁 + 카테고리 표준 | 섹션 구성·도메인 관행 (비슷이 갈지/차별화할지 판단) |
| **다른 도메인 내러티브 강자** | 도메인 무관, 스토리텔링/카피 흐름이 잘 짜인 랜딩 | 내러티브 arc·카피 흐름 차용 |
| **부분 UI 참고** | 특정 컴포넌트/섹션 (히어로·갤러리·CTA) 단위 | 시각 treatment 디테일 |

## 작성 규칙

각 슬롯에 **최소 2개 이상** 레퍼런스 + URL + *왜 이 슬롯에 들어가는지* 한 줄.

## 외부 리서치 의무 — 회고/추정 금지

**"타겟 visual diet" + 글로벌 트렌드 슬롯은 외부 검색 데이터 의무**.

회고/내 지식/추정으로 채우면 즉시 anti-pattern (#7). 검증 없는 단정은 거짓 일반화로 직결됨 — "AI 톤은 보라/파랑 dominant" (사실 Anthropic=코랄, OpenAI=그린, Vercel=흑백, Linear만 보라) 같은 사례.

### 추천 출처

**한국 청소년 visual diet**:
- 와이즈앱 — 앱 MAU·사용시간 통계
- 한국언론진흥재단 — 청소년 미디어 이용 보고서
- KISDI — 정보통신정책연구원
- Korea Herald — 트렌드 기사
- 마케팅 인사이트 / 오픈서베이 — 사용자 조사

**예시 데이터** (2026 기준, 매번 재검색):
- 한국 10대 visual diet 진짜: 인스타(MAU 412만, 압도) · 유튜브(사용시간 1위) · 틱톡(203만) · 카카오톡 · 콴다 · 핀터레스트(174만) · 클래스카드
- "토스/무신사/디스코드/카카오웹툰"는 *추정*이었음 — 데이터에 없거나 약함

**글로벌 디자인 트렌드**:
- Creative Bloq, Fireart, Smashing Magazine
- Awwwards, Land-Book, SiteInspire (큐레이션)
- 2026 흐름 예: anti-AI tactile 부상 (claymation, watercolor, 인간 손길) / Gen Z fun and real > professional / chaos and colors > perfection

```

출처: 와이즈앱 2025 모바일 앱 사용 분석 (URL)

- 콴다 — 자체
```

## 레퍼런스 분석 패턴

레퍼런스를 *분리해서 나열만* 하면 인사이트 없음. 다음 3단계 의무:

### 1) 공통 인사이트 추출

레퍼런스 2개 이상에서 *공통점*을 뽑음.

> 예: 카카오웹툰 표지 + Spotify Wrapped → 공통: *임팩트 있는 콘텐츠 placeholder가 자기 자리를 차지*. (카카오웹툰=일러스트, Spotify=데이터+typography)

### 2) 우리 제품 적용 옵션 변환

공통 인사이트를 *우리 제품 맥락*으로 옮김. 옵션 2-3개 제시.

> 예: 우리 학생 랜딩 placeholder 옵션:
> (a) 학생 일상 영상 (3초 모먼트)
> (b) 학습 데이터 시각화 (타이포 dominant)
> (c) 컨셉 일러스트 (단일 이미지)

### 3) 결정 시점 명시

옵션 중 *어느 시점에 결정할지* 명시.

> "production에서 콘텐츠 확보 후 결정" / "Phase 4 에셋 단계에서 결정"

**금지**: "X는 이렇고 Y는 저렇고" 분리만 하고 끝. 인사이트 추출 없으면 레퍼런스 가치 X.

## 안티 패턴

- ❌ 한 슬롯만 채우기 (예: visual diet만 보고 도메인/내러티브/UI 빼먹기)
- ❌ 슬롯 표시 없이 "예쁜 사이트들" 모음
- ❌ Selection bias — 디자이너 본인이 익숙한 사이트만. **타겟 명시 후 그 타겟의 visual diet 명시적 scan 강제**
- ❌ "현대적", "트렌디" 같은 모호한 정당화로 슬롯 채움
- ❌ **외부 리서치 0 — 회고/추정으로 채움** (anti-pattern #7, 가장 잦은 실수)
- ❌ 레퍼런스 분리 나열만, 공통 인사이트 추출 누락
